Search Data

There are four ways to engage with this interface:
Search Option 1. Search by Event Place (e.g. Magdalena, etc.) and optionally filter by year of the event.
Search Option 2. Search by Person name and optionally filter by year.
Search Option 3. Search by Event Type (e.g. baptism events, marriage events, etc.) and optionally filter by year of the event.
Search Option 4. Search by a person's Tribal Affiliation and optionally filter by year of the event the person is associated with.
Search Option 5. Combined search.

To select and search multiple event types at once, simply click on multiple options from the list below. Click on the option again to unselect.
